# Desafio Relâmpago

Projeto desenvolvido com foco em design limpo e elegante, oferecendo uma experiência visual agradável e funcional.

## Tecnologias Utilizadas

- **HTML5**: Estruturação semântica do conteúdo.
- **CSS3**: Estilização avançada, layout e animações.
- **JavaScript**: Interatividade e lógica de funcionamento.

## Funcionalidades e Design

- **Design Limpo e Elegante**: Interface moderna, minimalista e focada na usabilidade.
- **Responsividade**: O layout se adapta perfeitamente a diferentes tamanhos de tela (desktop, tablet e mobile).
- **Modo Claro/Escuro**: Alternância entre temas claro e escuro para maior conforto visual.
